# To-Do List Application

## Overview
A lightweight task management web application built using JavaScript, HTML5, and CSS3. This project demonstrates dynamic DOM manipulation, client-side state management, and interactive UI development.

Live Demo: https://fariha09.github.io/to-do-list/

## Features
- Add new tasks
- Edit existing tasks
- Mark tasks as completed
- Delete tasks
- Dynamic UI updates without page reload
- Responsive design

## Technologies Used
- JavaScript (ES6+)
- HTML5
- CSS3
- GitHub Pages (Deployment)

## Technical Highlights
- Implemented event-driven architecture for task interactions
- Managed application state on the client side
- Used DOM manipulation to dynamically update task list
- Structured modular and readable code for maintainability

## What I Learned
- Managing UI state efficiently in vanilla JavaScript
- Handling event listeners for dynamic content
- Improving user experience through responsive layout design

## Future Improvements
- Add local storage for persistent task saving
- Implement filtering (Completed / Active tasks)
- Add drag-and-drop functionality
